# Conformidade LGPD — índice

Ponto único para os documentos de privacidade/conformidade da plataforma. Tudo aqui é
versionado no git (o histórico guarda cada alteração) e revisado pela skill `concluir-tarefa`
ao fechar tarefas que tocam dados pessoais.

| Documento | Para quê |
|---|---|
| [ropa.md](ropa.md) | **ROPA** — inventário de tudo que o sistema faz com dados pessoais (Art. 37). |
| [subprocessadores.md](subprocessadores.md) | Parceiros que recebem dados, quais dados e em que país. |
| [resposta-incidentes.md](resposta-incidentes.md) | Passo a passo se houver vazamento/incidente (Art. 48). |
| [lgpd-e-conta.md](lgpd-e-conta.md) | Conta de acesso + autoatendimento de privacidade (exportar/excluir, retenção, Termos/Privacidade). |

**Documentos públicos (no app):** Política de Privacidade (`/privacidade.html`) e Termos
(`/termos.html`). **Encarregado (DPO):** `atendimento@nymbuslab.com.br` — resposta em até 15 dias.

**Relatório de conformidade:** rode `/lgpd-checker` no projeto para gerar/atualizar o relatório
(achados por categoria e severidade). Para entregar à ANPD/cliente, abra o documento e exporte em PDF.
